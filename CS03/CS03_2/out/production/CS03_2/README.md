# 💯 OX 퀴즈

## 🛠️ 작업 과정

- BufferedReader를 이용해서 문자열을 입력받고 정수로 변환해 테스트 케이스 변수에 저장합니다.
- 테스트 케이스 개수 만큼 반복문을 돌도록 설정하고 매 반복문마다 퀴즈의 결과인 문자열을 입력받습니다.
- 

## ⌨️ 실행 방법

- 첫째 줄에 테스트 케이스 개수를 입력합니다.
- 테스트 케이스 개수만큼 'O'와 'X'를 사용하여 공백없이 한줄씩 입력합니다.
