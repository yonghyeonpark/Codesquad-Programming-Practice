## 🛠️ 작업 과정

- BufferedReader를 이용해서 문자열을 입력받습니다.
- 문자열을 char 형으로 바꿀 수 있게 charAt을 사용합니다.
- 아스키 코드값을 출력하기 위해 다시 int로 형 변환을 하여 출력합니다.

## ⌨️ 실행 방법

- 알파벳 소문자, 대문자, 숫자 0-9 중 하나를 입력합니다.
